## Семейство Claude 4

Anthropic предлагает три линейки моделей, различающихся по мощности, скорости и стоимости.

### Claude Opus 4.6

Флагманская модель. Максимальная производительность на сложных задачах: рассуждение, кодирование, анализ, мультиагентные системы.

| Параметр | Значение |
|----------|----------|
| Model ID | `claude-opus-4-6` |
| Контекст (стандартный) | 200K tokens |
| Контекст (beta) | до 1M tokens |
| Максимальный выход | до 128K tokens |
| Input pricing (200K) | $5.00 / 1M tokens |
| Output pricing (200K) | $25.00 / 1M tokens |
| Input pricing (1M beta) | $10.00 / 1M tokens |
| Output pricing (1M beta) | $37.50 / 1M tokens |
| Prompt cache read (200K) | $0.50 / 1M tokens |
| Prompt cache write (200K) | $6.25 / 1M tokens |
| Дата выпуска | 5 февраля 2026 |

**Ключевые возможности:**
- Extended thinking (расширенное мышление)
- Adaptive thinking с уровнями effort 1-5
- Tool use (вызов функций)
- Tool search (экономия контекста при большом количестве инструментов)
- Vision (анализ изображений и PDF)
- Prompt caching
- Streaming

**Бенчмарки:**
- Terminal-Bench 2.0: 65.4%
- MRCR v2 (8-needle): 93% at 256K

### Claude Sonnet 4.6

Баланс между производительностью и стоимостью. Подходит для большинства задач, где не требуется максимальная глубина рассуждений.

| Параметр | Значение |
|----------|----------|
| Model ID | `claude-sonnet-4-6` |
| Контекст | 200K tokens |
| Максимальный выход | до 64K tokens |
| Input pricing | $3.00 / 1M tokens |
| Output pricing | $15.00 / 1M tokens |

**Примечание:** точные характеристики Sonnet 4.6 требуют уточнения по официальной документации. Приведенные данные -- ориентировочные, основаны на паттерне ценообразования Anthropic.

### Claude Haiku 4.5

Быстрая и дешевая модель для простых задач, классификации, фильтрации.

| Параметр | Значение |
|----------|----------|
| Model ID | `claude-haiku-4-5-20251001` |
| Контекст | 200K tokens |
| Максимальный выход | до 8K tokens |
| Input pricing | $0.80 / 1M tokens |
| Output pricing | $4.00 / 1M tokens |

**Примечание:** характеристики Haiku 4.5 требуют уточнения по официальной документации. Model ID включает дату (20251001), что типично для не-latest моделей.

## Model ID и версионирование

Anthropic использует два формата Model ID:

1. **Latest alias:** `claude-opus-4-6` -- всегда указывает на последнюю версию модели данного класса
2. **Dated version:** `claude-opus-4-6-20260205` -- фиксированная версия, привязанная к конкретной дате

Для production рекомендуется использовать dated version, чтобы избежать неожиданных изменений поведения модели при обновлении.
